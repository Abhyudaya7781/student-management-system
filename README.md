# 🎓 Student Management System (Java + SQLite)

A **Java project** that acts like a mini-DBMS using **SQLite + JDBC**.  
Supports full **CRUD operations**.

## 🚀 Features
- Add Student  
- View All Students  
- Search Student  
- Update Student  
- Delete Student
